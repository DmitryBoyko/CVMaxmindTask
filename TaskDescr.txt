Краткое описание
Разработать web-сервис для определения географического местоположения пользователя по IP адресу.

Требования:
1. Сервис должен хранить географические данные для всех IP-адресов в собственной базе
под управлением PostgreSQL.
2. Сервис должен регулярно обновлять указанную базу, используя данные какого-либо
поставщика (например, MaxMind GeoLite2: https://dev.maxmind.com/geoip/geoip2/geolite2/).
3. Сервис должен иметь REST API для получения географического местоположения (в
формате JSON) по заданному IP адресу.
4. Задача обновления базы данных должна быть реализована в виде консольного
приложения.
5. REST API должно быть реализовано на ASP.NET Core или ASP.NET Web API 2.
6. Обе указанные части сервиса должны быть написаны на C#.
